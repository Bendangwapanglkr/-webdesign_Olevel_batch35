Web browser: A web browser is a software or a type of application, that allow us to explore multiple websites. Example - Firefox and Google chrome. Web pages: Web pages are the pages with text, images and links inside a web browser. Websites: A website includes different web pages that we can access inside a web browser, while the website addresses stays the same. Webserver When we want to make a website accessible to everyone, we use webserver to store the required datas and send it to other users to view when they visit our website
